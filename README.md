# Smart-Timing (Full Stack)
Frontend: React + Vite + Bootstrap → **Vercel**  
Backend: Express + PostgreSQL → **Render**

Node.js v22.16.0 · npm 10.x

## Deploy
**Backend**
- Root: `backend`
- Build: `npm install`
- Start: `node server.js`
- Env: `DATABASE_URL`

**Frontend**
- Root: `frontend`
- Build: `npm run build`
- Output: `dist`

Frontend API → https://smart-timing.onrender.com/api  
CORS → https://smart-timing-git-main-daniel-qazis-projects.vercel.app
